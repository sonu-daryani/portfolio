# Sonu Daryani – Portfolio

A 3D animated portfolio built with **Vite**, **React**, **TypeScript**, **Three.js** (@react-three/fiber), and **Framer Motion**.

## Features

- 3D animated background (displacement sphere with custom shaders)
- Responsive layout with Tailwind CSS
- Smooth scroll and section animations
- Sections: Hero, About, Experience, Projects, Contact
- Links to LinkedIn, GitHub, IntalentAI, UnlockLife, winity.life

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Stack

- Vite 5
- React 18
- TypeScript 5
- Three.js + @react-three/fiber + @react-three/drei
- Framer Motion
- Tailwind CSS

## Type checking

```bash
npm run typecheck
```
